# Screenshots

This directory contains screenshots for the README documentation.

## Required Screenshots

1. **github-actions-success.png** - GitHub Actions workflow showing successful test run
2. **openapi-docs.png** - Interactive OpenAPI documentation at /docs
3. **user-registration.png** - User registration endpoint demonstration
4. **calculation-crud.png** - Calculation CRUD operations
5. **docker-hub.png** - Docker Hub repository with published images

## How to Add Screenshots

1. Take screenshots of the application running
2. Save them in this directory with descriptive names
3. Screenshots will automatically appear in the README

## Screenshot Guidelines

- Use PNG format for best quality
- Ensure sensitive information is redacted
- Capture full browser window or relevant section
- Use consistent resolution (1920x1080 recommended)
